# Student Management System 🎓

A web-based Student Management System designed to simplify and automate student-related administrative tasks such as student registration, attendance tracking, course management, academic records, and performance monitoring. This project provides an efficient and user-friendly platform for educational institutions to manage student data securely and effectively.

---

## 🚀 Features

* 👨‍🎓 Student Registration & Management
* 📚 Course & Subject Management
* 📝 Attendance Tracking
* 📊 Marks & Performance Management
* 🔐 Secure Authentication System
* 🧾 Student Record Management
* 📂 Dashboard for Admin/Staff
* ⚡ Responsive and User-Friendly Interface

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/tejayy/Student-Management-System.git
cd Student-Management-System
```

### 2. Create Virtual Environment

```bash
python -m venv venv
```

Activate it:

#### Windows

```bash
venv\Scripts\activate
```

#### macOS/Linux

```bash
source venv/bin/activate
```

---

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4. Apply Migrations

```bash
python manage.py migrate
```

---

### 5. Run the Server

```bash
python manage.py runserver
```

Open in browser:

```bash
http://127.0.0.1:8000/
```
